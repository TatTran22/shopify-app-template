# Shopify App Template

## Technologies:

- Backend: Laravel, Postgres, Redis
- Frontend: React with Inertia, Vite

## Installation

```zsh
yarn
```

```zsh
composer install
```

```zsh
cp .env.example .env
```

### Development using Docker

```zsh
cd web && docker compose up -d
```

At the root project

```zsh
yarn dev
```
